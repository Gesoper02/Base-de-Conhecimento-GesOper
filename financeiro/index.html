Conciliação Bancária - Processo Detalhado

Visão Geral do Processo

A conciliação bancária no GesOper utiliza uma interface dividida em dois painéis principais, permitindo comparação visual e eficiente entre os dados bancários e os registros do sistema.

1. Configuração de Período

Defina o intervalo de datas para análise. O sistema permite filtros flexíveis por data de movimento ou competência.

2. Importação de Dados

Importe arquivos OFX diretamente do banco. O sistema processa automaticamente os dados e os exibe no painel esquerdo.

3. Processamento

Execute a pesquisa para carregar os lançamentos do sistema no painel direito e iniciar o processo de comparação.

4. Conciliação

Compare e associe os lançamentos entre os painéis, resolvendo divergências e criando ajustes quando necessário.
