Sistema de Conciliação Bancária

Ferramenta para validação e controle de lançamentos bancários no sistema GesOper

O que é Conciliação Bancária?

A conciliação bancária é o processo de comparação entre os lançamentos registrados no sistema GesOper e os movimentos apresentados no extrato bancário, identificando divergências e garantindo a consistência dos dados financeiros.

Como Funciona o Sistema

1

Definir Período

Configure as datas inicial e final nos campos de período para delimitar o escopo da conciliação

2

Importar Arquivo OFX

Faça upload do arquivo OFX obtido diretamente do banco para carregar os dados do extrato

3

Executar Pesquisa

Clique em "Pesquisar" para carregar e processar os dados bancários no sistema

4

Comparar e Conciliar

Analise os painéis esquerdo e direito para identificar e resolver divergências

Estrutura dos Painéis

Painel Esquerdo - Extrato Bancário

  • Exibe movimentações importadas do arquivo OFX
  • Mostra data, histórico e valores dos lançamentos bancários
  • Permite seleção de lançamentos para conciliação
  • Indica status de conciliação de cada movimento
Ações Disponíveis:
Selecionar Lançamentos Filtrar por Data Ordenar por Valor

Painel Direito - Sistema GesOper

  • Apresenta lançamentos registrados no sistema
  • Exibe contas a pagar, receber e movimentações internas
  • Permite associação com lançamentos bancários
  • Mostra diferenças e pendências
Ações Disponíveis:
Associar Lançamentos Criar Ajustes Gerar Relatórios

Tipos de Conciliação

Conciliação Automática

O sistema identifica automaticamente correspondências exatas entre valores, datas e históricos.

Critérios:
  • Valor idêntico
  • Data compatível
  • Histórico similar

Conciliação Manual

Permite ao usuário associar manualmente lançamentos que não foram identificados automaticamente.

Quando usar:
  • Diferenças de histórico
  • Valores aproximados
  • Datas próximas

Ajustes e Correções

Criação de lançamentos de ajuste para resolver diferenças identificadas na conciliação.

Situações:
  • Tarifas bancárias
  • Juros e encargos
  • Diferenças de arredondamento

Situações Especiais

Lançamentos Não Localizados

Movimentos que aparecem no extrato bancário mas não possuem correspondência no sistema.

Solução: Verificar se o lançamento foi registrado corretamente ou criar lançamento de ajuste.

Diferenças de Valores

Lançamentos com valores divergentes entre o sistema e o extrato bancário.

Solução: Investigar a origem da diferença e criar lançamento de correção se necessário.

Múltiplas Correspondências

Um lançamento bancário que pode corresponder a vários lançamentos do sistema.

Solução: Analisar cada possibilidade e fazer a associação correta manualmente.

Filtros e Controles Disponíveis

Filtros por Período

  • Data inicial e final
  • Período pré-definido (últimos 30 dias, mês atual, etc.)
  • Filtro por data de movimento ou competência

Filtros por Status

  • Lançamentos conciliados
  • Lançamentos pendentes
  • Lançamentos com divergência

Filtros por Valor

  • Faixa de valores
  • Apenas débitos ou créditos
  • Valores acima de determinado limite

Dicas Importantes

Frequência

Realize a conciliação bancária diariamente para manter os dados sempre atualizados e identificar divergências rapidamente.

Arquivo OFX

Sempre utilize arquivos OFX atualizados, obtidos diretamente do internet banking do seu banco.

Verificação

Confira sempre os lançamentos não conciliados e investigue as causas das divergências encontradas.

Backup

Mantenha backup dos arquivos OFX utilizados para futuras consultas e auditorias.

Importante: A conciliação bancária é um processo fundamental para a gestão financeira. Mantenha sempre os dados atualizados e resolva as pendências o mais rápido possível.
